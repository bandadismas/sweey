## 🖥️ Getting Started

1. Clone the repository
   ```bash
   git clone https://github.com/bandadismas/sweey.git
   ```


2. Install dependencies
   ```bash
   pnpm install
   ```

3. Run the development server
   ```bash
   pnpm run dev
   ```
